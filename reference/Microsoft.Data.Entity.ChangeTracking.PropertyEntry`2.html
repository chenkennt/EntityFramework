Class PropertyEntry<TEntity, TProperty>

Provides access to change tracking information and operations for a given property. Instances of this class are returned from methods when using the ChangeTracker API and it is not designed to be directly constructed in your application code.

OriginalValue

Gets or sets the value that was assigned to this property when it was retrieved from the database.
