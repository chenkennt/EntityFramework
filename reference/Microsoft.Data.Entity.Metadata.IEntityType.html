ClrType

Gets the CLR class that is used to represent instances of this entity. Returns null if the entity does not have a corresponding CLR class (known as a shadow entity). Shadow entities are not currently supported in a model that is used at runtime with a DbContext. Therefore, shadow entities will only exist in migration model snapshots, etc.

FindPrimaryKey()

Gets primary key for this entity. Returns null if no primary key is defined. To be a valid model, each entity type must have a primary key defined. Therefore, the primary key may be null while the model is being created, but will be present by the time the model is used with a DbContext.
