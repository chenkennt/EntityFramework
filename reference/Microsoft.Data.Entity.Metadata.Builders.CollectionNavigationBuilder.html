Class CollectionNavigationBuilder

Provides a simple API for configuring a relationship where configuration began on an end of the relationship with a collection that contains instances of another entity type. Instances of this class are returned from methods when using the ModelBuilder API and it is not designed to be directly constructed in your application code.
